# iba-spec

**Intent Bound Authorization (IBA) Specification**

This is the formal spec for the governance layer on top of x402.

### What it covers
- Intent certificate format (cryptographically signed)
- Hard spending bounds (payees, caps, categories, expiration)
- Shard tokens for safe sub-agent delegation
- Binding to x402 payment flows
- WitnessBound integration for audit logs
